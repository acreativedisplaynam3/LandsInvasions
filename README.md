# LandsInvasion

LandsInvasion is a simple plugin with a few useful commands for Minecraft servers.

# Commands

- `/announce <text>` - Broadcasts a message as a title to all online players.



- `/discord` - Sends a link to your Discord server. 



- `/live <platform> <link>` - Broadcasts a message with the platform and link to your livestream.



- `/store` - Sends a link to your Minecraft server store. 

# Placeholders

- `%broadcast_player_name%` - Used in the config file to display the username of the player livestreaming content through the `/live` command.

- `%channel_link%` - Used in the config file to display the link to the livestream through the `/live` command.
